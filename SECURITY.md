# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | ✅        |

## Reporting a vulnerability

Please **do not** open a public issue for security-sensitive reports.

Instead, email the maintainer via GitHub profile contact or open a private security advisory:

**https://github.com/mythkiven/AudioLivenessDetection/security/advisories/new**

We will acknowledge receipt within 7 days and work on a fix or mitigation plan.

## Scope

This library processes PCM audio locally on device. It does not transmit audio to remote servers by itself. Integration code that forwards detection results to your backend is outside this repository's scope.
